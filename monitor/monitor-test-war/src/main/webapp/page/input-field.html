Test input field

Activating the buttons will log a request with the input field values. The buttons are sending requests with the HTTP method equal to the buttons label. The requests can be found in the requests page, use the text 'test/get' for the 'Get'-button, use the text 'test/put' for the 'Put'-button, use the text 'test/delete' for the 'Delete'-button, use the text 'test/head' for the 'Head'-button, use the text 'test/post' for the 'Post JSON'-button and use the text 'test/form' for the 'Test Post Form'-button in the Url field.
