Test alert as string

Test if an alert passed as a string with the event is placed in the HTML element with the ngc-alert and without the ngc-alert-target attribute. Activating the buttons will show an alert for info in the top container. It is shown in the style of info because it is the default level for alerts.
This is for convenience, so you can quickly send alerts with the event without having to create a Javascript object.
